# Mijia 插件 · 快速操作手冊 (v1.1)

> **核心提示**：本插件主打**自然語言控制**。絕大多數情況下，您不需要查找裝置 ID 或技術參數，直接用口語指令即可控制裝置。

---

## 一、 核心指令速查表

只需對 AI Agent 說出以下類型的指令，插件會自動解析並執行：

- **💡 開關裝置** — `打開臥室燈` / `關掉客廳插座`  
  支援「打開/關閉/開/關」等動詞
- **🏠 區域控制** — `打開客廳的燈` / `關閉主臥冷氣`  
  **(新版重點)** 支援「房間名+裝置名」
- **🌡️ 調節溫度** — `冷氣調到 26 度` / `冷氣設定 24 度`  
  自動辨識數字與單位
- **☀️ 調節亮度** — `燈調到 50%` / `檯燈亮度 30`  
  支援百分比或整數
- **🔄 切換模式** — `冷氣切換製冷` / `電風扇切自動`  
  支援中文模式詞
- **🎬 執行場景** — `執行回家模式` / `觸發離家場景`  
  需在小米 App 中預先設定好場景

## 二、 進階技巧：解決「裝置太多找不到」的問題

如果您家裡有多個同類型裝置（例如多盞「燈」），直接使用 `打開燈` 可能會觸發消歧提示。

### 1. 消歧提示範例
當指令不明確時，插件會回傳類似以下的提示，請依提示**再次補充房間名**：

> 找到 3 個符合「燈」的裝置：
> 1. 🟢 客廳 吸頂燈
> 2. 🔴 臥室 吸頂燈
> 請使用房間名+裝置名精確指定，例如「臥室燈」

### 2. 推薦的精確指令格式
為了避免混淆，建議養成使用 **`[房間名] + [裝置名]`** 的習慣：

*   ❌ 模糊指令：`打開燈`
*   ✅ 精確指令：`打開臥室燈`
*   ✅ 精確指令：`打開客廳的吸頂燈`

## 三、 常用 API 入口 (供開發者/進階使用者)

如果您需要直接呼叫插件介面，以下是核心入口說明：

- **`smart_control`** — 自然語言統一控制 (推薦)  
  參數：`{ "command": "打開臥室燈" }`
- **`list_devices`** — 取得裝置清單  
  參數：`{ "home_id": "12345" }`
- **`query_device_state`** — 查詢裝置狀態  
  參數：`{ "name": "冷氣" }`
- **`list_scenes`** — 取得智慧場景清單  
  參數：`{ "home_id": "12345" }`

## 四、 常見問題 (FAQ)

**Q: 為什麼提示「未登入」？**
A: 請先在 NEKO 插件面板中找到 Mijia 插件，點擊「掃碼登入」並完成授權。

**Q: 為什麼裝置離線但插件顯示在線？**
A: 插件顯示的是雲端狀態，可能存在延遲。請檢查裝置是否真的通電並連線到小米 App。

**Q: 支援哪些裝置？**
A: 理論上支援所有小米 IoT 平台裝置。只要能在小米 App 中控制的裝置，本插件均可支援。
